# Milestone 57.5: Adaptive Communication

## Architecture
Decouples reasoning from presentation. `AdaptiveSynthesizer` formats verified facts using templates for `EXECUTIVE`, `ENGINEER`, `RESEARCHER`.

## Reasoning Flow
Facts generated by LLM -> Critic -> Verifier -> Synthesizer styles output.
LLM rewriting is strictly reserved for `TEACHER` and `STUDENT` modes.

## Limitations
Heuristic detection of `CommunicationMode` can occasionally misclassify nuanced conversational prompts.

## Future Extensions
Personalized style profiles stored persistently in user settings.
