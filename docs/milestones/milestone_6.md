# Milestone 6 - Reviewer Recommendation Engine

Status: Completed

## Objective

Transform expertise knowledge into actionable reviewer recommendations.

The system should answer:

Who should review this change?

## Implemented Components

### Decision Layer

Introduced a dedicated decision layer.

Files:

decision/

* reviewer_recommendation.py
* reviewer_recommendation_service.py

---

### Recommendation Policies

Introduced pluggable recommendation strategies.

Files:

decision/policies/

* recommendation_policy.py
* coverage_recommendation_policy.py

---

### ReviewerRecommendation

Represents a recommendation generated by PIA.

Fields:

* reviewer_ref
* score
* covered_files

---

### ReviewerRecommendationService

Coordinates recommendation generation.

Responsibilities:

* Collect expertise information
* Delegate ranking to policies
* Return recommendations

The service itself contains no recommendation logic.

---

### CoverageRecommendationPolicy

First recommendation strategy.

Algorithm:

For each changed file:

* Retrieve experts

Aggregate by developer:

* Sum expertise scores
* Count covered files

Rank by:

1. covered_files
2. score
